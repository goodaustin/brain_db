🔥 Key Themes from This Batch

1. Memory Supercycle 全面啟動

DRAM 價格 YoY 4x (2Q25→2Q26), DDR5 合約 2Q26 再漲 50-60%。HBM 需求 2025→2027 翻 3.3x (2.3→7.6 bn GB)。全球 memory CapEx 到 2028 逼近 US$200bn。Micron BVPS FY28 >$300 (currently 1.3x P/B)。但 smartphone shipments 可能 <1bn units，memory 漲價正在擠壓手機 BOM cost。

2. CoWoS Capacity = 軍備競賽

TSMC CoWoS 2026→2027: 105→140K WPM。ASE/SPIL: 130→180K WPM。每 10K WPM 需要 10-12 台檢測設備 → 倍利科/牧德直接受惠。牧德半導體營收佔比從 14% (2026) 跳到 61% (2027)。倍利科 25-27 淨利 CAGR 74.7%。

3. Rubin 時代 PCB Content 爆發

PCB content per GPU: GB300 $443 → VR200 $623 → VR300 $2,534 (5.7x)。CCL content: $143 → $976 (6.8x)。LPU rack midplane = $100K PCB content。Rubin small patch 已開始 (March)，MP kick-off late April/May。EMC 材料連續兩輪漲價。CCL lead time 從 5-6 weeks 暴增到 >20 weeks。PCB TAM 2025→2027: RMB 28.9→163.7bn (5.7x)。

4. Smartphone 寒冬 vs AI 轉進

MS 下修 Android smartphone shipments 20-30%, global <1bn units。MediaTek 砍 20-30K 4nm wafers (15-20M chips)，但把產能轉向 Google TPU (2026e 3.6K → 2028e 7K units)。UMC 產能利用率持續下滑到 70%。DRAM/NAND 漲價進一步擠壓手機 BOM（8GB+256GB 從 $27 漲到 $173）。

5. 自動化復甦信號

China March PMI 50.4 (expansion)。HIWIN lead time 從 2-2.5M 延長到 3-4M，漲價 7-15%，MS 升評到 OW。AirTAC 目前 20x PE vs mid-cycle 25x。Humanoid robot 是潛在 upside risk。

6. Rubin Ultra 架構變更

從 4-die 改為 2-die + (2-die)×2。HBM4E 64GB/cube。Rubin Ultra 2-die = 512GB。CoWoP 用在 2×2-die 版本。ZDT (4958.TW) 被點名為 content gainer。
